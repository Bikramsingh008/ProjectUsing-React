YummyFood is a single-page food delivery web application built using React and React Router. It features a modern, component-based architecture with key sections like Home, About Us, Our Food, and Sign In. The app includes dynamic cart functionality using React's useState to manage item count, simulating real-world e-commerce cart behavior. The routing system enables seamless navigation without page reloads, enhancing user experience. The UI is modular with reusable components like Header and Footer, ensuring consistency across the site.
